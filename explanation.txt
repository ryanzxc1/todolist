도커 올린 후 수정
백
1. 자바 빌드(JAR생성)
cd back
./gradlew clean build -x test
2. 최상위 폴더로 이동:
cd ..
3. 도커 이미지 빌드 및 실행
docker-compose up --build

# 한 번에 실행하기
cd back && ./gradlew clean build -x test && cd .. && docker-compose up --build
